## string

### 子串substr的用法
```cpp
string s = "0123456789"; 
string a = s.substr(1,5);  // a = "12345"
string b = s.substr(1);    // b = "123456789"
```

### 删除字符
```cpp
string s = "0123456789";
s.erase(0, 2);  // 删除从下标0开始的连续两个字符，s="23456789"
s.erase(5);  // s = "23456",删除从下标5开始的全部字符
```

### 字符大小写转换
```cpp
char a = 'a';
a = toupper(a); // 大写
a = tolower(a); // 小写
```

### string转int
```cpp
string s = "12345"; 
int i = stoi(s);  // i = 12345
```
